Forklaring til opgaven:
Til at lave opgaven har jeg anvendt Ajax. Ajax er et sæt af webudviklings teknologier, som anvender mange webteknologier på client siden
for, at skabe en asynkron webapplikation. Dette betyder at webapplikationen kan sende og modtage data fra en server uden,
at interfere med selve displayet og det igangværende på den eksisterende side.
Vi adstiller derfor data udvekningslaget fra præsentationslaget og kan nu ændrer contentet dynamisk uden behov for, at reloade hele siden.

Som tidligere nævnt er anvender Ajax mange webteknologier, heraf:
# HTML og CSS til præsentation
# DOM (Document Object Model) for dynamisk visning og interaktion med data
# JSON (JavaScript Object Notation) til udveksning af data
# fetch() til asynkron kommunikation
# JavaScript til at sammensætte ovenstående teknologier

Jeg har derfor lavet denne opgave ved, at ved at oprette en div og button ved brug af HTML/CSS/DOM.
Derefter har jeg oprettet en function getJoke inde i index.js, som anvender fetch til at hente vores json.
Dernæst har jeg gennem javascript tilføjet en eventhandler på den oprettede knap inde i index.html,
samt tilføjet den getJoke functionen, således at en joke bliver vist, når man klikker.

NOTE: Nederste linje inde i funktionen getJoke ses det, at data.joke bliver hentet og er navnet på selve joken(quote).
Altså, det vil være muligt, at skrive data.id og så vil id'et komme istedet, når man klikker.
